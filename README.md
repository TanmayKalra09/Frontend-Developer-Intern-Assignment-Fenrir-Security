## Fenrir Security – Frontend Assignment
This project is a frontend implementation built as part of the Fenrir Security frontend assignment.  

## Tech Stack
- Next.js
- React
- Tailwind CSS
- Lucide React (Icons)
- ShadCN UI Components
- Sonner


## Features Implemented
- Responsive dashboard UI
- Interactive buttons and event handling
- Toast notifications:
- Functional components using React hooks
- Clean component structure
- Modern UI with proper spacing and alignment
- Lucide icons implementation


## Getting Started (Run Locally)
Follow these steps to run the project locally:
- [git clone <your-repository-url>](https://github.com/TanmayKalra09/Frontend-Developer-Intern-Assignment-Fenrir-Security.git)
- npm run dev
